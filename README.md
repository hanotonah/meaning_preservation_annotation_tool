# potato_meaning_preservation
Annotation tool created for a meaning preservation task for the Master thesis of Hannah Ottenschot (2024-2025) for the University of Twente. The tool is based on Potato: The Portable Annotation Tool (https://github.com/davidjurgens/potato). 

# Installation steps:
- Fetch this repository to your local machine 
- Create a new .venv
- Install Potato using `pip install potato-annotation==1.2.2.4`
- Replace the Potato folder in the .venv (.venv/Lib/site-packages/potato) with the version on this repository

Tested using Python version 3.12.6.

Once .venv has been installed and Potato files have been replaced, start the tool locally using the command `potato start meaning_preservation`. The terminal will show where the tool is running (`localhost:9001`). To access the tool, copy this link to your browser.

The first page is a login page. Create a new account (does not need to be an existing e-mail) to continue to the meaning preservation task.
